# Weather Rest Api

Rest API using https://www.weatherapi.com/

The project includes the following business logic:
1. CRUD for operations with city in database
2. Retrieving current weather
3. Forecast of weather for multiple days

- [x] Spring Boot
- [x] PostgreSQL
- [x] Swagger UI
- [x] Flyway
- [x] Log4j
- [x] Testing
